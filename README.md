# Estadistica para Negocios Internacionales — 2026-III

**Pontificia Universidad Javeriana**
Facultad de Ciencias Economicas y Administrativas
Profesor: Jaime Polanco Jimenez

## Contenido del curso

El curso cubre estadistica aplicada usando datos reales del ICFES (Saber 11 y Saber Pro), con enfasis en el programa de Negocios Internacionales.

### Unidades

| Unidad | Tema | Sesiones |
|--------|------|----------|
| U1 | Estadistica descriptiva y probabilidad | S01-S03 |
| U2 | Estimacion e intervalos de confianza | S04-S05 |
| U3 | Pruebas de hipotesis | S06-S10 |
| U4 | Regresion lineal | S11-S13 |

## Estructura del repositorio

```
sesiones/          Slides (PDF) y codigo R para cada sesion
problem_sets/      Enunciados (PDF) y plantillas de solucion (R)
proyecto/          Plantillas para las 3 fases del proyecto semestral
examenes/          Guias de repaso para parciales y final
datos/             Instrucciones para descargar los datos ICFES
utils/             Script de instalacion de paquetes R
```

## Software requerido

- [R](https://cran.r-project.org/) (>= 4.3)
- [RStudio](https://posit.co/download/rstudio-desktop/)
- Paquetes R: ejecutar `utils/setup_paquetes.R`

## Datos

Los datos provienen de las pruebas ICFES y deben descargarse por separado.
Ver `datos/README.txt` para instrucciones.
